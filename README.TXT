*   Le jeu comporte deux modes différents : le mode casual qui est le mode de jeu normal et le mode hardcore qui est le mode de jeu difficile.



*   Le jeu est jouable en qwerty et en azerty.



*   Toutes les cartes sont générées aléatoirement.
    Les cartes étant générées aléatoirement il se peut qu'une partie de celles-ci ne soit pas accessible. Il y a cependant forcément un chemin vers la sortie (case verte).



*   En jeu, le joueur peut quitter la partie à tout moment en entrant 'e' comme déplacement.



*   Explications des modes de jeu :
        - Mode Casual : se joue en entrant au clavier le mouvement désiré. Si plusieurs mouvements sont entrés ils seront exécutés en même temps.
        - Mode hardcore : se joue en entrant au clavier tous les mouvements nécessaires pour atteindre la sortie en une seule fois. Si le joueur n'atteint pas la sortie il a perdu.



*   Lorsque le joueur essaye d'aller sur une case occupée par un mur, il perd.



*   Légende :
        - Case bleue : Octocat
        - Case verte : Sortie
        - Case jaune : Case accessible
        - Case rouge : Mur



*   Il se peut que la taille par défaut du terminal ne suffise pas à afficher la carte et le nom du mode en cours.
    Il faut donc agrandir sa hauteur un petit peu pour que tout s'affiche correctement.
